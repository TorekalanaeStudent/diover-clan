## 6️⃣ Write Simple Use Case Statements

- The user can update their profile.
- The user can post listings of their product.

## 8️⃣ Identify the Most Important Use Cases

1. *User Registration & Login*

- Sign up with school email
- Login / Logout with session management

## Profile includes/User dashboard:
- Name
- Course & Year Level (MUST)
- Profile Photo (OPTIONAL)
- Contact Number (MUST)

2. *Post an Item for Sale/MEssenger Type of Shop*

- Photo of the Item
- Set Title, Description, price, and Category
- Choose item condition like (new, used, etc.) 
- Set meetup location preference
- Option to tag item as Fo Sale or For Swap

3. *Browse & Search Listings*

- View all active listings on frontpage
- Search items by keyword
- Filter by category 

Sort By:
- Price (Low to High / High to Low)
- Newest First
- Location / Meetup Spot

4. *Item Details & Inquiry*

- View full item information and photos
- View seller profile and ratings
- Send a message directly to the seller
- Save / Bookmark items for later viewing

5. *Manage Listings*

- Edit item details or update price
- Makr item as sold
- View all personal Listings

6. Rate & Reveiw After Transaction

- Buyer rates seller (1-5 starts + comment) and vice versa
